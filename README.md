# British-Airways-Passenger-CSAT-Dashboard
Developed an end-to-end Power BI analytics solution using British Airways customer review data. Performed data cleaning, modeling, and DAX calculations to create dynamic KPIs, parameter-driven metrics, geographic maps, trend analysis, and interactive slicers to evaluate customer satisfaction across aircraft, cabin class and service factors.

This project focuses on analyzing British Airways customer review data to understand passenger satisfaction and identify key factors that influence the overall travel experience. The main objective of this project is to transform raw review data into meaningful insights using Power BI, helping airline stakeholders make data-driven decisions to improve service quality and customer experience.

An interactive and dynamic dashboard is developed where users can explore customer satisfaction across different service metrics, aircraft types, cabin classes, travel types, and geographic locations.
